Introduction

Welcome to IdioSplash, an open-source solution for enterprise networks, facilitating Meraki iPSK & WPN onboarding through AWS, Cloudflare, and the Meraki Dashboard API.

What is IdioSplash

Welcome to IdioSplash, an open-source initiative engineered to enhance and secure network access within enterprise settings. Utilizing Cisco Meraki's Identity Pre-shared Key (IPSK) technology, IdioSplash introduces a distinctive gateway—a splash page, securely hosted on AWS and fortified by Cloudflare's robust security suite. With a simple email verification process, users are granted a unique IPSK code, enabling personalized access to the network. The open-source ethos of IdioSplash allows for a tailored approach, promoting a secure and personalized wireless network access experience.

At the core of IdioSplash's architecture is the combination of AWS and Cloudflare technologies. AWS Lambda effortlessly handles backend processing tasks such as OTP generation and verification, while Amazon DynamoDB securely stores data and triggers email notifications via Amazon SES. On the other side, Cloudflare Pages hosts our static website, ensuring a fast, secure, and user-friendly interface. The blend of AWS and Cloudflare not only creates scalable infrastructure but also provides a seamless platform for IPSK generation and management, offering reliability with user-centric design.

How It Works

Navigate through this section to discover the user and admin experiences, delve into the architectural backbone, and grasp the financial and operational facets of IdioSplash.

User Experience

Step 1: Navigate to IdioSplash Domain

Users head to ex. arcadia.idiosplash.com

Step 2: Email Verification

Enter their organization email and hit the 'Verify' button.

Step 3: Enter OTP Code

Check their email for a One-Time Passcode (OTP) and enter it in the OTP field.

Step 4: Receive Passcode

Their unique passcode is displayed and as a QR code for easy device onboarding.

Final Step: Connect and Forget

The unique IPSK password obtained is the key to a seamless network connection. Here's what's next:

  • Input IPSK Password: Enter this password into the Wi-Fi settings of their devices.
  • Automatic Connection: Once done, your devices will securely connect to the network.
  • Enjoy Seamless Connectivity: With the IPSK set, enjoy a hassle-free and consistent connectivity across all your devices!

Admin Experience

Admins will need to follow the installation and setup instructions provided in the IdioSplash documentation to get things up and running (including Meraki SSIDs and such). As well as customizing the service themselves by changing and writing additional code themselves.

Once the setup is complete, managing IPSKs is a breeze via the Meraki dashboard.

Expenses and Limitations

Limitations of Meraki iPSK

While the Identity Pre-Shared Key (IPSK) without RADIUS feature provides a robust solution for network security and management, there are certain limitations to be aware of when considering it as a solution for your network.

  • AP Compatibility: All APs must be Wi-Fi 5 Wave 2 or newer and running MR 27.1 or newer firmware.
  • Encryption Support: WPA3 encryption is not supported with IPSK without RADIUS.
  • iPSK Limit: Up to 50 iPSKs per SSID are supported in firmware versions MR 27.X, 28.X, and 29.X, extending up to 5,000 iPSKs per SSID in firmware versions MR 30.1 and newer.
